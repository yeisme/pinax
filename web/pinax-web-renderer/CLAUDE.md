# Pinax Static Renderer Instructions

This package is a static publish renderer for Pinax export bundles. It is not an internal workbench, standalone app, Vite page, Bun HTML app, or Electron preview shell.

## Commands

```bash
bun run test
bun run build
bun run render:static
```

## Boundaries

- Keep source as pure TypeScript renderer and Markdown pipeline code.
- Do not add React, Vite, page routes, HTML app entries, browser state, or local workbench UI here.
- Static HTML files generated by `render-static.ts` are user publish artifacts and may include `index.html`, `notes/**/index.html`, `tags/**/index.html`, and `pinax-data/**`.
- Do not write vault metadata, `.pinax/**`, credentials, raw provider payloads, private paths, raw prompts, or full chain-of-thought from this renderer.
